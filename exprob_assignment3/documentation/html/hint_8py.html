Detailed Description

This program elaborates the hints received.

Author
Zoe Betta
Version
1.0
Date
12/06/2022

Subscribes to:
None

Publishes to:
/complete

Services:
/hint /checkcomplete /result

Client Services:
armor_interface_srv

Action Services:
None

Description:
This node implements the server that receives the hint and elaborates it saving it in the ontology, the server returns true if the hint was well formed, false otherwise. It also implements the server to check if there is at least one complete and consistent hypothesis. ALso the server that given an ID retrieves the field of the hypothesis from the ontology.
